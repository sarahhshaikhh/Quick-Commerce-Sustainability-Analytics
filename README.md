# 📊 Quick Commerce Sustainability Analytics

> Business Intelligence, Machine Learning, and Game Theory for Sustainable Decision-Making in Quick-Commerce

## Research Objective

The objective of this project is to develop a Business Intelligence framework capable of:

* Evaluating warehouse performance using operational data
* Measuring sustainability through Economic, Environmental, and Social indicators
* Predicting warehouse profitability using Machine Learning
* Segmenting warehouse performance through clustering techniques
* Supporting strategic decisions using Game Theory

---

### 🌱 Environmental Sustainability

### 👥 Social Sustainability

## Methodology

The project follows a Business Intelligence workflow consisting of:

1. Data Collection
2. Data Cleaning & Preprocessing
3. KPI Engineering
4. Exploratory Data Analysis
5. Business Intelligence Dashboard Development
6. Machine Learning Modeling
7. Strategic Analysis using Nash Equilibrium
8. Interpretation of Business Insights

---

## Machine Learning

The following Machine Learning techniques were used:

* **Random Forest Regression** for warehouse profitability prediction
* **K-Means Clustering** for warehouse segmentation based on sustainability indicators

---

## Key Insights

The proposed framework enables organizations to:

* Monitor sustainability performance across multiple warehouse locations
* Identify high-performing and underperforming warehouses
* Analyze operational efficiency and environmental impact
* Evaluate workforce-related sustainability metrics
* Support strategic decision-making using analytical insights
